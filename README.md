# Campus-Life-Electronic-Mall
基于SSM的校园生活电子商城是专为校园师生打造的购物平台。采用Spring进行依赖注入和事务管理，SpringMVC负责请求分发与页面跳转，MyBatis完成数据库操作。该商城提供各类商品的展示、选购、下单及支付功能，还能根据校园场景进行个性化推荐，满足师生日常购物需求，为校园购物带来便利，营造便捷、安全、丰富的校园购物环境。
